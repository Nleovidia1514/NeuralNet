CD      [route][..]                 MOVE THROUGH SYSTEM DIRECTORIES
ECHO    [messagee][message:>:file1] SHOWS MESSAGES ON CONSOLE OR CREATES A TXT FILE WITH SPECIFIED MESSAGE
DIR      [no argument]               SHOWS A DETAILED LIST OF ALL THE FILES IN CURRENT DIRECTORY
COPY      [file1:dir][file1:file2]    COPIES A FILE FROM SOURCE TO DESTINATION
DEL      [file1:dir][dir]            DELETES A FILE OR A DIRECTORY
MOVE      [file1:dir][file1:file2]    RENAMES OR MOVES A FILE FROM ITS CURRENT POSITION TO THE SPECIFIED DIRECTORY
MKDIR   [dir]                       CREATES A NEW DIRECTORY
RMDIR   [route][dir]                REMOVES A DIRECTORY AND ALL ITS CONTENTS
DATE    [no argument]               SHOWS SYSTEM DATE AND TIME
